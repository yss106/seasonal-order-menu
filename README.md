# 好时味 · 时令点菜

一个无需构建工具的响应式点菜 Web 小程序。它会根据当前月份自动切换春、夏、秋、冬主题与时令推荐菜品。

## 本机运行

直接双击 `启动点菜小程序.bat`，或在本目录打开 PowerShell，执行：

```powershell
python -m http.server 8080
```

然后在浏览器打开 `http://localhost:8080`。

## 手机打开

确保手机和电脑连接同一个 Wi-Fi。双击启动文件后，它会显示手机访问地址。

当前网络下的示例地址：

```text
http://172.20.10.2:8080
```

若 Windows 防火墙弹出提示，需要允许 Python 在专用网络通信。

## 修改菜单

在 `app.js` 的 `seasons` 中修改四季菜品，在 `commonDishes` 中修改全年供应菜品。
